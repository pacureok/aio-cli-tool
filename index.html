Guía de Uso Rápido

AIO CLI Tool es una herramienta de línea de comandos todo en uno. Aquí te mostramos cómo usar sus funciones principales desde la terminal.

Cómo Funciona el Comando

Una vez instalado, puedes usar el comando aio.exe desde cualquier lugar de tu terminal. Los comandos básicos son:

  • **aio.exe --help**: Muestra todos los comandos y opciones disponibles.
  • **aio.exe build [archivo]**: Compila un archivo .aio en un formato ejecutable.
  • **aio.exe process [archivo]**: Procesa un archivo de idioma o metadatos para su uso.

Archivos y su Uso

AIO CLI Tool asocia varios tipos de archivos a su funcionalidad. Cada extensión de archivo tiene un propósito específico:

Archivos .aio

Estos archivos son el núcleo de la herramienta. Contienen código fuente que se compila para crear un ejecutable. Su ícono de archivo está asociado a la función de **construcción**.


; Ejemplo de código dentro de un archivo .aio
; Define un nuevo proyecto
Project: MiAplicacion
Version: 1.0

; Agrega archivos fuente
Source: main.go
Source: utils.go

; Configura las dependencias
Dependencies:
    - github.com/ejemplo/libreria

Para compilarlo, usa el comando:


aio.exe build "C:\ruta\a\MiAplicacion.aio"

Archivos .esp e .ing

Estos archivos contienen datos para la traducción a español e inglés, respectivamente. El ícono de cada archivo está asociado a la función de **procesamiento**.


; Ejemplo de un archivo .esp
welcome_message=Bienvenido a AIO
error_message=Ha ocurrido un error

Para procesar el archivo, usa el comando:


aio.exe process "C:\ruta\a\textos_espanol.esp"

Archivos .meta

Estos archivos almacenan metadatos para tu proyecto, como información de autor o versión. Su ícono de archivo está asociado a la función de **procesamiento**.


; Ejemplo de un archivo .meta
Author: Juan Perez
Date: 2024-08-17
Description: Herramienta de línea de comandos para proyectos AIO.

Archivos .crea, .go y .rs

Estos son los archivos de código fuente que se usarán en tu proyecto. Los íconos se asocian a la función de **procesamiento**.


// Ejemplo de un archivo .crea
function init_project() {
    // Código para inicializar un nuevo proyecto
}
