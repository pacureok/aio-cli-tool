Guía de Uso Completa

AIO CLI Tool es una herramienta de línea de comandos para compilar y procesar tus archivos. Esta guía te ayudará a empezar de forma rápida y sencilla.

1. La Terminal (CMD)

La terminal es tu "consola" o "pantalla de comandos". No es un video, es un programa donde escribes comandos de texto. Aquí es donde usarás aio.exe.

  • Para abrirla, busca **CMD** en la barra de búsqueda de Windows y presiona Enter.
  • Para cerrar la terminal, puedes escribir el comando exit y presionar Enter, o simplemente hacer clic en el botón **X** de la ventana.

Cómo Usar los Comandos

Una vez instalado, el comando aio.exe funcionará desde cualquier carpeta. Aquí están los comandos principales que necesitarás:

  • **aio.exe --help**: Muestra todos los comandos y opciones disponibles. Es tu "manual" de uso.
  • **aio.exe build [ruta_del_archivo]**: Compila un archivo .aio en un programa ejecutable.
  • **aio.exe process [ruta_del_archivo]**: Procesa archivos de datos para que la herramienta pueda leerlos.

2. Tipos de Archivos y sus Usos

AIO CLI Tool asocia varios tipos de archivos a su funcionalidad. Haz doble clic en ellos para que la herramienta los procese automáticamente o úsalos con los comandos en la terminal.

Archivos .aio

Son el núcleo de la herramienta. Contienen el código fuente del proyecto y están asociados a la función de **construcción (build)**. Al hacer doble clic en ellos, se ejecutarán para construir tu proyecto.


; Ejemplo de un archivo .aio para construir un proyecto
Project: MiAplicacion
Version: 1.0
Source: main.go
Source: data.rs
Dependencies:
    - github.com/ejemplo/libreria

En la terminal, el comando para compilarlo es:


aio.exe build "C:\ruta\a\MiAplicacion.aio"

Archivos .esp e .ing

Contienen los datos de traducción a español e inglés. Cuando se usan en el proyecto, la herramienta los procesa para que tu aplicación sea multi-idioma.


; Ejemplo de un archivo .esp para traducción a español
welcome_message=Bienvenido a la aplicación
error_message=Ha ocurrido un error

En la terminal, el comando para procesarlo es:


aio.exe process "C:\ruta\a\textos.esp"

Archivos .meta

Almacenan metadatos del proyecto, como información de autor o versión. Estos datos son cruciales para el proceso de compilación y se procesan automáticamente.


; Ejemplo de un archivo .meta para metadatos del proyecto
Author: Tu Nombre
Date: 2025-08-17
Description: Herramienta de línea de comandos para proyectos.

Archivos de Código Fuente (`.crea`, `.go`, `.rs`)

Estos son los archivos de código fuente que se usarán en tu proyecto y también están asociados a la función de **procesamiento**. La herramienta sabe cómo leer y usar el código que contienen para la construcción final del programa.
