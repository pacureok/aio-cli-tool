Guía de Uso Rápido

AIO CLI Tool es una herramienta de línea de comandos para compilar y procesar tus archivos. Una vez instalado, puedes usar el comando aio.exe desde cualquier lugar de tu terminal.

Comandos Básicos

  • **aio.exe --help**: Muestra todos los comandos y opciones disponibles.
  • **aio.exe build [archivo]**: Compila un archivo .aio.
  • **aio.exe process [archivo]**: Procesa un archivo de idioma o metadatos.

Tipos de Archivos y sus Usos

AIO CLI Tool asocia varios tipos de archivos a su funcionalidad. Cada extensión tiene un propósito específico:

Archivos .aio

Son el núcleo de la herramienta. Contienen el código fuente del proyecto y están asociados a la función de **construcción (build)**.


; Ejemplo de un archivo .aio
Project: MiAplicacion
Version: 1.0
Source: main.go
Source: data.rs
Dependencies:
    - github.com/ejemplo/libreria

Para compilarlo desde la terminal:


aio.exe build "C:\ruta\a\MiAplicacion.aio"

Archivos .esp e .ing

Contienen datos de traducción a español e inglés. Están asociados a la función de **procesamiento**.


; Ejemplo de un archivo .esp
welcome_message=Bienvenido
error_message=Ha ocurrido un error

Archivos .meta

Almacenan metadatos del proyecto, como información de autor. También asociados a la función de **procesamiento**.


; Ejemplo de un archivo .meta
Author: Tu Nombre
Date: 2025-08-17
Description: Herramienta de línea de comandos.

Archivos de Código Fuente (`.crea`, `.go`, `.rs`)

Son los archivos de código fuente que se usarán en tu proyecto y están asociados a la función de **procesamiento**.
